# Along the River During the Qingming Festival 3D

可於瀏覽器直接運行的互動式 3D《清明上河圖》教育場景，重現北宋汴京、汴河、虹橋、碼頭、市集、城門及古代城市生活。

## 第一版功能

- 動態汴河水面、船隻與水波
- 橫跨河面的木製虹橋
- 北宋風格木建築、瓦頂、商店、茶館、市集及城門
- 多種會行走及擺動的人物，包括行人、挑夫、官員及守城士兵
- 馬車、牛車、狗及家禽
- 電影式自動導覽及自由遊覽模式
- 支援桌面、平板與手機
- 不需要安裝套件，直接使用 GitHub Pages 運行

## 操作

- 拖曳：轉動視角
- 滾輪／雙指：拉近、拉遠
- W A S D／方向鍵：自由遊覽時移動
- Shift：加速
- 介面按鈕：暫停導覽、切換自由遊覽、開關環境聲及調整畫質

## 技術

- Three.js
- WebGL
- 原生 JavaScript ES Modules
- 程序生成建築、人物、船隻、動物及動畫

## 運行

直接開啟 GitHub Pages，或下載後用靜態伺服器：

```bash
python -m http.server 8080
```

然後進入 `http://localhost:8080`。

## 下一階段

目前是可運行的程序生成原型。要提升至真正電影級，可以逐步加入 Blender 製作的 GLB 模型、骨骼人物、PBR 材質、碰撞尋路、LOD、群眾 AI、體積霧及 WebGPU 後期效果。
